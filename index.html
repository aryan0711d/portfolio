Projects

Psychological Horror Prototype

A first-person psychological horror prototype focused on environmental transformation and tension-driven gameplay. The project features a custom cycle-based system where the world evolves as players uncover fragmented memories and new gameplay rules. Core mechanics include a sanity system, dynamic lighting transitions, spatial 3D audio cues, and adaptive post-processing. The project was optimized using occlusion culling, static batching, and mixed lighting workflows to maintain strong performance while preserving atmosphere.
